# Sunrise

Continuous Integration / Continuous Deployment Workshop by Truss Works.

## Usage

Start Docker cluster.

```shell
make up
```

Stop Docker cluster.

```shell
make down
```

Run End-To-End JavaScript tests.

```shell
make e2e_tests
```

Run unit tests.

```shell
make unit_tests
```

Run server tests.

```shell
make server_tests
```
